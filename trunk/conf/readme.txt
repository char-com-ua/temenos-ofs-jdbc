This directory contains sample files to configure t24 libraries:


channels.xml                    the t24 connection channels configuration

t24jdbc.xml                     the mapping file to translate LOCALREF field names 
                                to readable names in error messages.
                                this file also could contain "no records found message" detector.

tcclientlog.properties          example configuration of the log4j for tcclient
                                this configuration is compatible with wso2 servers.
